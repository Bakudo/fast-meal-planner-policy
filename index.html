Privacy Policy

1. General Information

This Privacy Policy describes how the Fast Meal Planner application ("Application") collects, uses, and protects user data. By using the Application, the user agrees to the terms described in this document.

2. What Data Do We Collect?

The Application itself does not collect or store personal data outside the user's device. However, we use third-party tools that may collect certain information:

  • Google Analytics for Firebase – collects anonymous statistical data about Application usage (e.g., usage time, feature interactions, errors).
  • Firebase Crashlytics – collects information about Application crashes to help us improve stability.
  • Internet Connection – The Application may require internet access to download or save data but does not transmit any private user information.

3. How Do We Use the Collected Data?

The collected data is used exclusively for the following purposes:

  • Analyzing Application usage and optimizing its functionality.
  • Improving performance and stability.
  • Diagnosing errors and crashes.

4. Data Sharing

We do not sell, rent, or share user data with third parties. Data collected by Google Analytics and Crashlytics is processed in accordance with Google's privacy policy: https://policies.google.com/privacy.

5. Permissions Required by the Application

The Application may require access to:

  • Internet to synchronize data and access online resources.
  • Internal Storage (if applicable) – to store user data locally.

6. Security

We take user data security seriously and implement appropriate technical measures to protect information. The Application does not store any passwords or sensitive data.

7. Changes to the Privacy Policy

We reserve the right to update this Privacy Policy. Changes will be published in the Application or on our website.
